# 🚢 Titanic Survival Prediction

## 📌 Overview
This project builds a Machine Learning model to predict whether a passenger survived the infamous Titanic disaster. It demonstrates core data science workflow steps, including data preprocessing, feature engineering, and predictive modeling.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Scikit-Learn
* **Algorithm:** Logistic Regression

## 🚀 Key Features & Workflow
* **Data Preprocessing:** Cleaned the raw dataset by handling missing values and encoding categorical features.
* **Model Training:** Split the data into training and testing sets, and trained a Logistic Regression model.
* **Performance:** The model successfully achieves an accuracy of **78.09%** on the test dataset.
